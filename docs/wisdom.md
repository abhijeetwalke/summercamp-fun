# wisdom.md — The Distilled Why

The soul of the project in 16 lines. Everything else is detail. If a decision ever conflicts with this file, this file wins — flag it to Abhi.

1. **This is a gift.** A father building a rich learning experience for his 11-year-old son — and possibly his son's friends. Build everything to that standard: with care, never cheap.
2. **Diagnose before teaching.** Surface skill gaps first; all teaching is keyed to exposed gaps. This is a diagnostic engine, not a tutoring product.
3. **Pull, don't push.** Easy early wins create the hook; difficulty ramps to meet skill (flow). He should come back on his own — never be dragged back.
4. **Reward effort, never ability.** Never "you're smart." Celebrate what the work produced: first-click streaks, training completed, territory earned.
5. **No shame, ever.** Mistakes feed the loop. Bounce-backs — solving a fresh variant of an old miss — are celebrated as proof the brain rewired. Clean, positive, age-appropriate space throughout.
6. **Clicks-to-correct is the truth.** One honest raw signal over motivational summaries. The data is visible to him, not hidden from him.
7. **Hints guide, never carry.** A nudge, then a first step, then nothing — the full thinking model does the teaching afterward, not the hint.
8. **The real work happens in the notebook.** This is not a clicking game. Hard multi-step problems are wanted; the screen only records the result.
9. **Mastery is earned and visible.** 80% first-click turns a territory green and a bender into an Adept — he's building a world, not watching a number.
10. **Content is never cheap.** 800 competition-caliber questions, each with a thinking-model solution and two derived hints — validated, tagged, spot-checked. Spend the time and tokens; no sloppiness.
11. **Theme is a wrapper, rigor is the core.** Stories and journeys motivate; the problems stay rigorous. Never let the skin soften the math.
12. **Depth-first.** Perfect one subject completely before the next moves — no breadth temptations. *(Knowingly overridden once: Abhi chose to build World Awareness alongside Math on 2026-06-17 — a deliberate, logged exception, not a drift. The principle still stands for everything else.)*
13. **Agile beats pretty.** Never make him wait. Any effect that costs load speed or responsiveness loses.
14. **Sessions have an ending.** ~45 bounded minutes; no endless screen time. The Mission ends, the day's win is banked.
15. **Structure over numbers.** Counts, pool sizes, ratios, schedules are config — tuned from real data, not sacred values.
16. **Nothing lands unseen.** Discuss → finalize → explicit approval → preview → "looks good?" → main. Tasks count as done only when they're on `main`. (Full workflow: CLAUDE.md.)
